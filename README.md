# Basic UNIX Shell

A simple, yet functional command-line shell for a UNIX-based system that can execute commands, handle basic built-in functions, and provide a basic user interface.



## Features
1) Command Execution
    - `ls`, `pwd`, `echo`
2) Built-in Commands
    - `cd`, `exit`, `help`
3) Prompt Display
    - `user@user ~ $`
4) Input Handling
5) Process Management
    - `fork()`, `exec()`, `wait()`
6) Signal Handling
   - `ctrl + c`
7) Command History
